# 🏦 Sistema Bancário Java

Projeto desenvolvido com Java e MySQL com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO), JDBC, SQL e arquitetura em camadas, simulando um sistema bancário semelhante aos utilizados por instituições financeiras.

## 🚀 Tecnologias

- Java
- JDBC
- MySQL
- IntelliJ IDEA
- Git
- GitHub

## 📂 Arquitetura

O projeto foi organizado em camadas para separar responsabilidades.

```
src
│
├── app
├── database
├── model
├── repository
└── service
```

## ✅ Funcionalidades

- Cadastro de clientes
- Abertura de contas
- Login utilizando CPF e senha
- Depósito
- Saque
- Transferência entre contas
- Consulta de saldo
- Extrato bancário
- Registro de movimentações no banco de dados

## 📚 Conceitos aplicados

- Programação Orientada a Objetos
- Encapsulamento
- Classes e Objetos
- Relacionamento entre entidades
- JDBC
- CRUD
- SQL
- Repository Pattern
- Service Layer
- Separação de responsabilidades
- Persistência de dados com MySQL

## 🎯 Objetivo

Este projeto faz parte da minha evolução como desenvolvedor Back-end Java.

O foco é construir um sistema cada vez mais próximo de um ambiente profissional, aplicando boas práticas de desenvolvimento e arquitetura de software.
